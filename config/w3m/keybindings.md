# w3m Keyboard Shortcuts

## Basic Navigation

| Key | Action |
|-----|--------|
| `j` / `↓` | Move down one line |
| `k` / `↑` | Move up one line |
| `h` / `←` | Move left |
| `l` / `→` | Move right |
| `gg` | Go to top of page |
| `G` | Go to bottom of page |
| `0` | Go to beginning of line |
| `$` | Go to end of line |

## Scrolling

| Key | Action |
|-----|--------|
| `Space` | Scroll down one page |
| `Ctrl+d` | Scroll down half page |
| `Ctrl+u` | Scroll up half page |
| `Ctrl+f` | Scroll down full page |
| `Ctrl+b` | Scroll up full page |

## Tab Management

| Key | Action |
|-----|--------|
| `t` | Open current link in new tab |
| `T` | Open new tab (prompts for URL) |
| `d` | Close current tab |
| `gt` | Go to next tab |
| `gT` | Go to previous tab |
| `W` | Show tab menu (view all tabs) |

## Links & Navigation

| Key | Action |
|-----|--------|
| `Tab` | Move to next link |
| `Shift+Tab` | Move to previous link |
| `Enter` | Follow current link |
| `[[` | Go to previous link |
| `]]` | Go to next link |
| `f` | List all links on page |
| `B` | Go back to previous page |
| `o` | Open URL (goto) |

## Search

| Key | Action |
|-----|--------|
| `/` | Search forward |
| `?` | Search backward |
| `n` | Next search result |
| `N` | Previous search result |

## Bookmarks & History

| Key | Action |
|-----|--------|
| `m` | Add current page to bookmarks |
| `M` | View bookmarks |
| `H` | View history |
| `q` | Exit from menus/history/bookmarks |

## Page Actions

| Key | Action |
|-----|--------|
| `r` | Reload current page |
| `R` | Reshape/reformat page |
| `v` | View HTML source |
| `s` | Select buffer/page menu |

## Clipboard

| Key | Action |
|-----|--------|
| `y` | Copy current link URL to clipboard |
| `C` | Copy current link URL to clipboard |

## Help & Misc

| Key | Action |
|-----|--------|
| `?` | Show help |
| `q` | Quit w3m (or exit menus) |
| `Q` | Quit without confirmation |
| `!` | Execute shell command |

## Color Scheme

- **Links**: Green
- **Active/Selected Link**: White text on green background
- **Images**: Yellow
- **Forms**: Cyan
- **Marked Text**: Magenta

## Features Enabled

- Smooth scrolling
- Mouse support
- Inline image display
- UTF-8/Unicode support
- Cookie management (accepts cookies, blocks bad ones)
- Link numbering
- Line info display
- Case-insensitive search
- Persistent history (last 1000 URLs)
- Persistent bookmarks
- Privacy protection (no referer sending)
- Downloads saved to ~/Downloads
